SUSPECT PROFILE CARDS — MURDER MYSTERY BOARD GAME
===================================================
PURPOSE: These 50 character profile cards are fully interchangeable across any case scenario.
The same card can represent a murderer, victim, witness, suspect, alibi provider, accomplice,
or bystander depending on the case being played. The "Relationship" tag is a default starting
flavor; the web app dynamically reassigns roles per scenario. Profiles are intentionally
written with ambiguity — details can point to guilt or innocence depending on context.
===================================================
